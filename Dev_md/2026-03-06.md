# 개발일지 - 2026-03-06

## DreamIT Biz Docs 프로젝트 초기 구축

### 개요
분야별 학습 자료를 정리하고 브라우저에서 PDF를 바로 열람할 수 있는 문서 라이브러리 사이트를 구축했다.

### 기술 스택
- **프론트엔드**: React 19 + React Router 7
- **빌드 도구**: Vite 6
- **배포**: GitHub Pages (GitHub Actions 자동 배포)
- **도메인**: docs.dreamitbiz.com

### 프로젝트 구조
```
D:\docs\
├── public/pdfs/          # 분야별 PDF 파일 저장소
├── src/config/site.js    # 사이트 설정 (메뉴, 카테고리, 회사정보)
├── src/data/documents.js # 문서 목록 데이터
├── src/pages/            # 홈, 카테고리, 뷰어, 404 페이지
├── src/components/       # Navbar, Footer, CategoryCard, PDFViewer 등
├── src/contexts/         # ThemeContext, ToastContext
├── src/styles/           # CSS 스타일시트 (12개 파일)
└── .github/workflows/    # CI/CD 자동 배포
```

#### 3. 주요 기능
- **PDF 바로보기**: iframe 기반 브라우저 내장 PDF 뷰어
- **문서 검색**: Ctrl+K 단축키, 제목/설명/태그 검색
- **테마 시스템**: Dark/Light/Auto 모드 + 5가지 색상 테마
- **반응형 디자인**: 데스크톱, 태블릿, 모바일 대응
- **AOS 애니메이션**: 스크롤 시 요소 등장 효과
- **Glassmorphism 네비바**: blur 효과 적용 고정 네비게이션
